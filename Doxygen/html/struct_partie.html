Detailed Description

Objet modélisant la partie de jeu Qwirkle.

L'objet Partie stocke les informations relatives à la partie de jeu tel que : le nombre de joueurs NbJoueurs, l'index du joueurCourant, le tableau de Joueur, le nombre de tuiles restantes dans le sac nbTuilesSac, le sac contenant les pointeurs sur les Tuiles, le lot de Tuile contante les Tuiles crées en statique de manière ordonée, le plateau sur lequel les joueurs pose leurs tuiles
