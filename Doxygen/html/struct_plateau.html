Detailed Description

Objet modélisant le plateau de jeu Qwirkle.

L'objet Plateau contient les bornes du plateau où les tuiles sont jouées ainsi qu'un tableau 2 dimensions des tuiles déjà jouées sur ce plateau les bornes xmin,ymin,xmax et ymax ne sont utilisés que pour la V0 sur la console
